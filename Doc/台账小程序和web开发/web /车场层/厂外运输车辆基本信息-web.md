# 1.查询条件

车牌、审核状态、下发状态、排放阶段；

# 2.grid

序号、车牌号码、号牌颜色、车辆类型、车辆识别代码(VIN)、注册登记日期、车辆品牌型号、燃料类型、排放标准、联网状态、使用性质、登记时间、修改时间、下发时间、审核状态、下发状态、次数、发动机号码、发动机型号、发动机制造商、核定载质量(kg)、准牵引质量/kg、手机号码、是否安装、车队名称、进厂运输货物名称、进厂运输量/吨、出厂运输货物名称、出厂运输量/吨、住址、发证日期；

车头照片、行驶证（照片）、随车清单照片在grid左侧；

审核状态：已审核、未审核；

下发状态：已下发、未下发；

# 3.下发：

已审核的车辆才支持下发；

支持批量下发、单个下发；
